# Nexus - Site Management Matrix

[English](#english) | [中文版](#中文版)

---

## English

Nexus is a lightweight, SaaS-grade site management and monitoring system. It features a modern user interface, fully responsive design, and robust role-based access control.

### Features
- Intelligent Probe & Crawler: Automated 30-minute polling to check site availability.
- Adaptive Dual-Theme Engine: Frosted glass UI for Light and Dark modes.
- Enterprise-Grade RBAC: Strict privilege isolation between SuperAdmin and Admin.

### Quick Start
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python3 app.py`

## 中文版

Nexus 是一个具备现代化 SaaS 级界面、响应式设计以及权限管理的轻量级站点监控与导航系统。

### 核心特性
- 智能探针与爬虫：全自动轮询测活，智能抓取目标站点内容。
- 昼夜双擎引擎：全局磨砂玻璃 UI，完美适配护眼模式。
- 企业级权限隔离：SuperAdmin 与 Admin 严格权限隔离。

### 快速部署
1. 克隆仓库。
2. 安装依赖环境：`pip install -r requirements.txt`
3. 启动系统：`python3 app.py`
